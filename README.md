# agile-admin

> 简介:这是一个敏捷后台，基于vue-admin-template模板二次开发而来，里面包括基础的后台模块，用户，角色，权限，导入记录，系统日志，可以用来快速开发模块，后端使用laravel8作为框架开发，地址：https://github.com/liaoyimin123/agile-admin-core

部署环境
node16.20.1 + npm8.19.4

部署流程
1.安装包
npm install

2.修改接口地址
根目录下.env.development文件

3.启动项目
npm run dev

4.项目打包
npm run build
